# phpbrew安装扩展时报openssl找不到
> configure: error: Cannot find OpenSSL's <evp.h>

## 指定PKG_CONFIG_PATH
```
export PKG_CONFIG_PATH=/usr/local/opt/openssl/lib/pkgconfig
```
