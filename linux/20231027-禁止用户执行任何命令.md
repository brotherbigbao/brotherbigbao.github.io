# ubuntu 禁止用户执行任何命令
> 禁止用户执行任何命令

# 设置用户默认bash

/etc/passwd

```
/bin/bash 改成 /bin/rbash
```

# 更改$PATH

在 /home/yourname/.profile中追加下面这行：

```
export PATH="/home/yourname"
```

# 修改 /home/yourname 下文件权限

```
chmod 444 .profile
chmod 444 .bashrc
```
